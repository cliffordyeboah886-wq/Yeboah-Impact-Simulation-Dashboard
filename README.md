# Yeboah-Impact-Simulation-Dashboard
An interactive computational framework for simulating mechanical impact efficiency based on Yeboah’s Law. This dashboard visualizes the relationship between tool mass, impact velocity, and material resistance to determine the Efficiency Index ( 𝜂 η), providing real-time empirical curve analysis for predicting material cleavage versus rebound.
